Step 2: Create Dockerfile

Step 3: Add Docker Hub credentials in Jenkins

Go to:

Jenkins Dashboard → Manage Jenkins → Credentials → Global → Add Credentials

Select:

Kind: Username with password
Username: your Docker Hub username
Password: your Docker Hub password/token
ID: docker-creds
Step 4: Create Jenkinsfile

Change this line:

IMAGE_NAME = "rahuljangra/my-app"

Use your Docker Hub username and image name.

Also change:

url: 'https://github.com/RahulJangra79/your-repo.git'
Step 5: Push project to GitHub
git init
git add .
git commit -m "add jenkins pipeline"
git branch -M main
git remote add origin https://github.com/RahulJangra79/your-repo.git
git push -u origin main
Step 6: Create Jenkins Pipeline Job

Go to:

Jenkins Dashboard → New Item

Enter name:

docker-ci-cd-pipeline

Select:

Pipeline

Click OK.

Step 7: Configure Pipeline

In Pipeline section:

Definition: Pipeline script from SCM
SCM: Git
Repository URL: your GitHub repo URL
Branch: */main
Script Path: Jenkinsfile

Click Save.

Step 8: Build Pipeline

Click:

Build Now

Jenkins will perform:

Clone GitHub repo
Build Docker image
Tag image with Jenkins build number
Login to Docker Hub
Push image to Docker Hub
Run container after successful push
Step 9: Verify container

On Jenkins machine, run:

docker ps

Open browser:

http://localhost:3000
